## 🔢 What's Inside

This folder is the complete analytical output for this match, generated by the **Critechery pipeline**:

- **Humdinger Score** — composite match excitement rating
- **Critechery Score** — team performance index (batting + bowling)
- **Burn Rate** — resource-space innings visualisation
- **Pressure Index** — heatmap of pressure across the match
- **BLACK MIRROR** — counterfactual "what if?" scenarios
- **Win Probability** — ball-by-ball with Volatility Index
- **Game Changers** — the players who decided the result

All metrics, ball-by-ball data, and chart PNGs are in this folder.
